# SQRBlock

SQRBlock is a blockchain-based QR-driven product traceability platform designed for secure product verification and supply chain transparency.

## Features
- QR-based product authentication
- Blockchain-backed traceability system
- Supply chain tracking workflow
- Product verification dashboard
- Real-time product lookup
- Flask-based backend system

## Technologies Used
- Python
- Flask
- Blockchain
- Solidity
- SQLite
- HTML/CSS
- QR Code Integration

## Project Structure
- `app.py` → Main Flask application
- `blockchain.py` → Blockchain logic
- `ProductTraceability.sol` → Smart contract
- `templates/` → Frontend HTML templates
- `static/` → Static assets

## How It Works
1. Product information is registered
2. QR code is generated
3. Blockchain stores traceability records
4. Users scan QR code for product verification
5. System displays product history and authenticity
